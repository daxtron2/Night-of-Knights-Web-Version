Night of Knights: Web Version

Welcome to the landing page for Night of Knights! This is a web game created by TJ Wolschon and Mike Schek for IGME-230 Web Design and Implementation. Night of Knights began as a class project in January 2017 for a different class. This version, which uses the assets and is inspired by that original project, has been created for the web using Phaser, a JavaScript framework for games. There are 3 different versions of the game that can be accessed on this site. Each with different changes to make them play in different ways.
